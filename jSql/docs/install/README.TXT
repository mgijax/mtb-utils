jSql

March 8. 2004

Installation
-------------------------------------------------------------------------------
This is a self extractable JAR file.  Simple execute java -jar jSql-Install.jar
to start the installation process.

Using
-------------------------------------------------------------------------------
Move the .jSql_properties file to your home directory.

Go to the directory to where the archive was unzipped and execute 
java  -jar jSql.jar

Notes
-------------------------------------------------------------------------------
Currently, JDBC drivers are distributed with the installer for Oracle, MySql,
and Sybase.
